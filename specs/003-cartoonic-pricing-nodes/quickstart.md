# Quickstart: Cartoonic Pricing & Tech Nodes

## Run locally
1. `npm install`
2. `npm run dev`
3. Open `http://localhost:3002`.

## Verify
- Switch to the Cartoon theme using the theme toggle.
- Confirm that "The AI Ecosystem" (Projects) and "Learning Journey" sections no longer use dark backgrounds.
- Confirm testimonial cards use lighter surfaces with readable text.
- Confirm the email pill/button in Contact has readable text and icon contrast.
- Ensure no layout shifts in the Pricing/Services grid.
